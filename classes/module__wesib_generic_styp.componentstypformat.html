Class ComponentStypFormat

Component style production format.

This format can be obtained from component context.

The formats implemented:

  • ComponentStypObjectFormat (the default) renders CSS using CSS object model.
  • ComponentStypDomFormat renders CSS as text. May render CSS of disconnected element.
