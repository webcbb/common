  • handle(__namedParameters: { context: ComponentContext<T>; event: Event; navigation: Navigation; page: Page }): void
    • Defined in wesib/generic/src/navigation/handle-nav-links.decorator.ts:80

    Handles event by performing navigation action.

    Every event sent by one of elements inside decorated component is passed to this function. In response it may perform a navigation event.

    By default handles events on anchor tags. When such tag contains an href attribute containing same-origin URL it opens a page at this URL instead of default action. It also prevents navigation if URL didn't change.
