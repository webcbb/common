Optional transfer

  • transfer(to: Page, when: "pre-open" | "pre-replace" | "enter" | "return"): Handle<T, I> | undefined
    • Defined in src/navigation/page-param.ts:102

    Transfers parameter to target page.

    This is called right before LeavePageEvent is fired for each parameter handle of current page.

    Parameters

    • to: Page

      A page to transfer parameter to.

    • when: "pre-open" | "pre-replace" | "enter" | "return"

      When the transfer happens. Either pre-open, pre-replace, open, or return. The latter is used when return to page generated by another app version. E.g. from the page that has been reloaded.

    Returns Handle<T, I> | undefined

    New parameter handle instance for target page, or undefined if nothing to transfer.
