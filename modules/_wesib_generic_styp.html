Component style producer function interface.

Type declaration

    • (rules: StypRules, opts?: ComponentStypOptions): EventSupply

    • Parameters

      • rules: StypRules

        CSS rules to produce stylesheets for. This can be e.g. a StypRule.rules to render all rules, or a result of StypRuleList.grab() method call to render only matching ones.

      • Optional opts: ComponentStypOptions

        Production options.

      Returns EventSupply

      Styles supply. Once cut off (i.e. its off() method is called) the produced stylesheets are removed.

    A decorator of component property returning CSS rules to produce.
