Converter definition of enclosing component's user input control.

The returned converted control keeper may send an event supply as a second parameter. This supply will be cut off once the input from converted control is no longer needed. Otherwise the control's input supply will be cut off instead, and control would become unusable after that.

Configures @ConvertInput component decorator.

DefaultInAspects

DefaultInAspects: AfterEvent<[InConverter.Aspect<any, any>]>
  • Defined in wesib/generic/src/input/default-in-aspects.ts:22

Default input aspects.

This is an AfterEvent keeper of aspect converter meant to be applied to controls.

As a bare minimum it assigns the following aspects to converted controls:

  • InRenderScheduler set to ElementRenderScheduler,
  • InNamespaceAliaser set to `DefaultNamespaceAliaser.

More input aspect converters may be registered in context. They may override the default ones.
